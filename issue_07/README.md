# Issue #7: The GLP-1 Gold Rush

## Overview

This folder contains the complete analysis for Issue #7 of *The American Healthcare Conundrum*: a quantitative projection of the CMS BALANCE Model (Broad Access and Lower Costs for Essential Drug Needs) for GLP-1 receptor agonists under Medicare Part D, 2027–2036.

**Headline findings:**
- **Eligible beneficiaries (MID scenario):** 13.2M Medicare beneficiaries (obesity rate 35.5% × engagement rate 55%)
- **10-year program cost:** $86.2B–$170.1B (range across LOW/MID/HIGH scenarios; baseline MID = $124.8B)
- **Peak annual enrollment:** 4.64M beneficiaries in 2032
- **Negotiated pricing:** $245–350/month (vs. $1,000–1,350/month retail) = 70% price reduction
- **Annual savings at full ramp:** $40.0B/year (range $40–80B)
- **Health benefit ROI:** 33% conservative ($41.3B benefits / $124.8B program cost), 56% moderate, 97% optimistic

## Files

### Analysis Pipeline

**`01_build_data.py`** — Self-contained Python script that implements the entire BALANCE projection model:
- Downloads: None required (all data hardcoded from published sources)
- Inputs: Medicare population, obesity rates, engagement benchmarks, negotiated prices (all cited)
- Outputs: Six CSV files + summary JSON (see below)
- Run as: `python3 01_build_data.py`

### Output Data (results/)

Generated by `01_build_data.py`:

1. **`balance_projection_all_scenarios.csv`** — 30 rows (10 years × 3 scenarios)
   - Year-by-year enrollment and cost tracking

2. **`sensitivity_analysis.csv`** — 6 rows (obesity ±2%, engagement ±5%, enrollment ramp fast/slow)
   - Model uncertainty drivers

3. **`health_benefit_roi.csv`** — 3 rows (Conservative, Moderate, Optimistic)
   - Cost-benefit analysis

4. **`international_prices.csv`** — 7 rows (US retail, BALANCE, peer countries)
   - Price comparison narrative

5. **`market_growth.csv`** — 8 rows (2018–2025 actual + projected)
   - Historical context and market trajectory

6. **`key_metrics.json`** — Summary JSON with all headline numbers, assumptions, caveats

## Methodology Highlights

1. **Eligible population:** Medicare beneficiaries × obesity rate × engagement rate
2. **Enrollment ramp:** 5% (2027) → 12% (2028) → 40% plateau (2031+)
3. **Retention:** 88% annual (attrition from mortality, discontinuation)
4. **Pricing:** CMS BALANCE negotiated rates ($245–350/month) vs. $1,000–1,350 retail
5. **Health benefits:** $1,190–$3,500 per person-year of treatment (conservative to optimistic)
6. **Market context:** $71.7B current (2023) → $95B projected (2025) under negotiation

All assumptions are cited with sources in the script header.

## Caveats

- Obesity rates are static (actual NHANES data are cross-sectional)
- Retention model simplified (real cohort attrition steeper year 1)
- Price durability assumes CMS enforcement and manufacturer compliance
- Generic entry (Ozempic patent 2030) not modeled; pricing may compress further
- Benefits use generic complication-prevention values; actual ROI varies by patient
